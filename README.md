# *知乎日报*
![日报icon](readmefile/icon2.png)
重庆邮电大学红岩移动开发部Android学员2018年寒假考核作业
## 应用简介
+ 实现API：1.splash sreen\
2.软件版本查询\
3.最新消息获取\
4.消息内容获取\
5.过往消息获取新闻额外信息（评论数量、点赞数量）\
6.长评论查看\
7.短评论查看
+ 6.0运行时权限，用相册图片改变头像
